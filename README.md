# Vehicles Management System

##  Project Overview
This project implements a **Vehicles Management System** using Java.  
It serves as a console-based application that allows users to manage information about vehicles, including adding, displaying, and updating vehicle records.

The system is designed with clear structure and organized functionality.

---

##  Key Features
- Add new vehicle entries  
- Display all vehicles  
- Search for specific vehicles  
- Update vehicle information  
- Clean and understandable code structure  

---

##  Technologies Used
- Java  
- Standard Java Libraries  

---

##  Main Components
- Vehicle class  
- Vehicle management operations  
- Main application logic  

Each part is structured for readability and maintainability.

---

##  How to Run

1. Clone this repository:https://github.com/MHAZXD/Vehicles.git
2. Navigate to the project folder: cd Vehicles
3. Compile all Java files: javac *.java
4. Run the main program: java Main
